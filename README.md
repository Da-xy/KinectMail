README :

Fonctionne avec Processing 3.3.3 et antérieure mais pas avec la dernière version 3.3.4 sortie le 3 juin



1) Dans le cas d'une connexion à GMAIL :

	- Il faut modifier un paramètre de sécurité pour permettre à une application de se connecter.

	- Pour ce faire, suivre ce lien : https://myaccount.google.com/security#activity

	- En bas à droite de la page, se trouve l'option --> Paramètre "Autoriser les applications moins sécurisées"

	- Il faut switcher cette option à "Activé" si ce n'est pas déjà fait.


2) Possibilité que l'antivirus ou le parefeu bloque la connexion, le désactiver dans ce cas là

3) Ne pas oublier de mettre la librairie javaxmail dans le dossier processing

4) Pour la fenêtre de connexion appuyer sur espace pour utiliser la souris plus pratique pour saisir les identifiants 
et re appuyer sur espace pour se mettre en mode kinect
